Champion Reads® Online Website

This project is a web companion for the Champion Reads® book series, which highlights athletes who have overcome adversity. The website and books are approved for use as classroom resources during instructional time because they align with the Indiana school curriculum. I contributed to the development of two games: “What Would You Do?” and “Eye Spy.” The first game asks users to respond to simple scenarios using the knowledge they gained from learning about each featured athlete. The second game challenges users to find relevant items in each athlete's childhood bedroom. Both games award points based on correctness and allow users to download a personalized report. They also incorporate generative AI image generation, customized to fit Champion Reads' unique environment. In addition to the games, I developed a VTT parsing algorithm that processes VTT files I created in English, Spanish, and French to provide live captions for the website's guided presentations. These captions support students who may not be able to play audio or who do not speak English. I also implemented various site-wide updates requested by the project owner, such as adding the District of Columbia to the Geography game and adjusting fonts and button colors. This project was built using HTML, CSS, and JavaScript.

Information Technology Help Desk Application

This C# project is an IT Help Desk website designed to manage support tickets with a functional login page and ticketing system that connects to a MySQL database. Users can submit, view, and track the status of their IT issues, while help desk agents can manage and resolve tickets. The project emphasizes collaboration within a team of five developers, providing hands-on experience in version control, task delegation, and software design principles. By integrating backend database functionality and user authentication, the project enhances my skills in full-stack development, database management, and team-based software engineering and this project can be found on my Butler Github.

Command Line Shape Jumper Game

A Linux console version of Geometry Dash inspired by RobTop Games. Shape Jumper challenges players to complete rhythm-based obstacle levels with precision and timing, all within a terminal interface. I developed this game engine from the ground up using C++, focusing on clean game logic and lightweight dependencies. To enhance the user experience, I integrated ncurses for terminal UI rendering and SFML for audio playback, delivering an engaging interactive experience without graphical assets. Key features include: Authentic rhythm-platformer gameplay adapted for a text-based interface Automatic progress saving and level persistence Debug and help runtime flags for enhanced testing and accessibility Built for Linux environments with make, this project improved my skills in game architecture, systems programming, and cross-platform tooling. Source code is available on GitHub, and prebuilt binaries are provided for easy installation and testing. Tech stack: C++, ncurses, SFML, Makefile Scope: Solo development, full game engine implementation
